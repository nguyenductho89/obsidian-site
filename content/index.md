# Welcome to Obsidian Site

This is a self-hosted [Obsidian](https://obsidian.md) publish site powered by [Quartz 4](https://quartz.jzhao.xyz/).

## Features

- **Wikilinks**: Link between notes using `[[note-name]]` syntax
- **Backlinks**: See which notes link to the current page
- **Graph View**: Visualize connections between notes
- **Full-text Search**: Press `Ctrl + K` to search
- **Tags**: Organize notes with #tags
- **Dark Mode**: Toggle between light and dark themes

## Getting Started

Check out the [[getting-started|Getting Started]] guide to learn how to add your own notes.
